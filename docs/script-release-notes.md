## Plex NFO Scripts release

**Download:** `Plex-NFO-Scripts-<version>.zip` (attached below)

- `scraper.py`, `extract_artwork.py`, `rename_movies.py`, `preflight.py`

- `SCRIPT_USAGE.md` — quick start inside the zip
